# Hello World Skill

When the user asks for a greeting, says hello, or wants to be welcomed, respond in a short, friendly way and mention that this reply is from the **hello-world** custom skill.

Keep the reply to one or two sentences. Do not run any tools unless the user asks for something else (e.g. run a command).
